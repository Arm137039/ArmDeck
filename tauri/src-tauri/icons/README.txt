Place the following icon files in this directory:

- 32x32.png (32x32 pixels)
- 128x128.png (128x128 pixels)
- 128x128@2x.png (256x256 pixels)
- icon.icns (macOS icon)
- icon.ico (Windows icon)

These icons will be used by the Tauri application for the window icon and taskbar/dock icon.
You can generate these icons from a single high-resolution image using tools like:
- https://appicon.co/
- https://www.npmjs.com/package/@tauri-apps/cli-darwin
- https://www.npmjs.com/package/@tauri-apps/cli-win32
